Tindakan Perbaikan:
1.	Pembersihan keseluruhan PC:
Pembersihan PC menggunakan blower untuk menghilangkan debu dan melap casingnya
2.	Diagnosa awal:
Pengecekan dengan menggunakan PSU lain untuk mengetahui apakah kedua ram berfungsi (512x2 ddr1). Hasil akhir salah satu ram rusak karena konektor atau kuningan pada ram terkena korosi
3.	Penggantian ram
Ram di ganti dari 512 ddr1 ke 1x2 ddr1
Hasil: lebih responsif, tetapi masih kewalahan di OS moderen
4.	Penggantian PSU
PSU lama di ganti dengan unit baru yang lebih stabil
Konektor lengkap + daya aman
5.	Penggantian OS
Instalasi Linux ringan (Antix)
Testing aplikasi ringan seperti browser ringan (Midori, Firefox)
Hasil Akhir:
PC berhasil boot ke OS ringan
Bisa dipakai untuk pengetikan, file, media ringan
Browser berat masih struggle, tapi usable
